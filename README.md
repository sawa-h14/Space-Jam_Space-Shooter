## Key Bindings
| Action  | Key Bindings |
| ------------- | ------------- |
| Thrust (Move foward) | W  |
| Drag (Move backwards) | S  |
| Turn Left  | A  |
| Turn Right  | D  |
| Nose Up (Turn up) | ↑ (Arrow Up) |
| Nose Down (Turn down) | ↓ (Arrow Down) |
| Tilt Left (Roll left) | Q  |
| Tilt Right (Roll right) | E  |
